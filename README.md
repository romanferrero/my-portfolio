# 🚀 Portfolio — Roman Ferrero

Portfolio personal construido con **React 19**, **Tailwind CSS 4** y **Framer Motion**. Diseñado como una single-page application con soporte bilingüe (ES/EN), dark mode, animaciones fluidas y arquitectura completamente data-driven.

🔗 **[Ver sitio en vivo →](https://romanferrero.com)** *(reemplazar con URL real)*

---

## ✨ Features

- **Bilingüe** — Español e inglés con detección automática del idioma del navegador (i18next).
- **Dark / Light mode** — Toggle de tema con persistencia en localStorage vía Context API.
- **Animaciones** — Transiciones y stagger animations con Framer Motion.
- **Data-driven** — Proyectos, skills, experiencia y links sociales se administran desde archivos de datos centralizados.
- **Scroll progress** — Barra de progreso de lectura en el navbar.
- **Accesibilidad** — Skip to content, semántica HTML y contraste optimizado.
- **SEO** — Sitemap, robots.txt y meta tags configurados.
- **Responsive** — Diseño adaptado a todos los tamaños de pantalla.

## 📄 Secciones

| Sección | Descripción |
|---------|-------------|
| **Hero** | Presentación con CTAs a proyectos, contacto y descarga de CV. |
| **Sobre mí** | Bio, estadísticas y trayectoria resumida. |
| **Skills** | Grilla de tecnologías agrupadas por categoría (frontend, backend, bases de datos, herramientas). |
| **Proyectos** | Cards con descripción bilingüe, tags y links a GitHub/demo. |
| **Experiencia** | Timeline combinado de educación y trabajo con detalle de cada etapa. |
| **Contacto** | Formulario de contacto y links sociales. |

## 🛠️ Stack

- **React 19** — Componentes funcionales, Hooks y Context API.
- **Tailwind CSS 4** — Styling utility-first vía plugin de Vite.
- **Framer Motion** — Animaciones declarativas y scroll-triggered.
- **i18next + react-i18next** — Internacionalización con detección automática de idioma.
- **Lucide React + React Icons** — Iconografía.
- **Vite 8** — Build tool y dev server.
